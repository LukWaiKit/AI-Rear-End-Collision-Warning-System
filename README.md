# AI-Rear-End-Collision-Warning-System
Overview
This repository showcases my contributions to a final-year team project at Lunghwa University of Science and Technology.

The project aims to develop an AI-based vehicle rear-end collision warning system using Raspberry Pi 5, Python, YOLOv8, and OpenCV. The system performs real-time vehicle detection, estimates the distance between vehicles, and provides an audio warning when the following distance falls below a predefined safety threshold.

Note: This repository is created for portfolio purposes. It showcases my individual contributions to a university team project. Some project components are not included because they were developed collaboratively or are not available for public release.
